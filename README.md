# About This App

## App Details

    This App is about online Laptop selling shop . In this App
    you will find some cards about names of Laptops and their
    prices . If you want to order any Laptop ,You have to Click
    on the card. This Process will redirect to Selling page. On
    that page if you click on Buy Now Button you will get
    confirmation message and get your laptop.

## Features

    1. Customer can select a laptop and order it.
    2. Implemented side scrolling card.
    3. Built in navigation.
    4. Easy to Use.
